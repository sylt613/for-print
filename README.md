# For Print

Hebrew Sefer PDFs ready for printing.

## Files

- **shortened_by_aron.pdf** - Main version
- **shortened_by_aron_onepage.pdf** - Single page layout  
- **shortened_by_aron_diff.pdf** - Diff version

## Format

All PDFs are formatted using the Sefer typography system with:
- Frank Ruehl Libre font
- First word bold and enlarged
- Second line indented
- Last line centered
- Full justification

Generated from LaTeX using sefer.sty package.
